# 🚀 Control System Simulator

A web-based Control System Simulator built using Python and Streamlit to analyze linear time-invariant (LTI) systems using transfer functions.

---

## 🌐 Live Demo

👉 https://csanalyser-xicqqq9lyshgratn2whawc.streamlit.app/

---

## 📌 Features

* 🔹 Transfer Function Input (Numerator & Denominator)
* 🔹 Step Response Plot
* 🔹 Impulse Response Plot
* 🔹 Bode Plot (Magnitude & Phase)
* 🔹 Root Locus Plot
* 🔹 Stability Analysis (Stable / Unstable)
* 🔹 Poles & Zeros Display
* 🔹 Performance Metrics:

  * Rise Time
  * Settling Time
  * Overshoot
* 🔹 Gain Margin & Phase Margin
* 🔹 Predefined Systems (First-order, Second-order, Underdamped)
* 🔹 Download Plots as Images
* 🔹 Clean and Interactive UI
